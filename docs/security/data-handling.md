# Data Handling Policy

How ARCHON handles user data — what we store, how long we keep it, and how to delete it.

---

## Data We Store

### What we store from your repository

ARCHON **never stores raw source code files**. When you connect a repository:

1. We clone the repository temporarily into worker memory
2. We split code into chunks and generate vector embeddings
3. We store **only the embeddings + chunk metadata** (file path, line numbers, language)
4. The raw code is discarded after indexing

**Chunk metadata stored:**
- File path (relative)
- Language
- Start/end line numbers
- Token count
- SHA-256 hash of content (for deduplication)

**Not stored:**
- Raw source code content
- File contents
- Commit history
- Author information

### What we store from review outputs

Review outputs (ADRs, IaC, diagrams, findings) are stored and associated with your account. These contain:
- Architecture findings generated by AI agents
- Terraform code generated by AI agents (not copied from your repo)
- Cited web sources and excerpts

### What we store for auth and billing

- Email address and name (from Clerk)
- GitHub username (if connected)
- Stripe customer ID and subscription status (no card numbers — Stripe handles payment data)

## Data Retention

| Data Type | Retention | Deletion Trigger |
|---|---|---|
| Code chunk embeddings | Until project deleted | Project deletion or account deletion |
| Review packages (ADRs, IaC, diagrams) | 90 days | Auto-deleted, or project/account deletion |
| Citations | 90 days | Same as review packages |
| Agent reasoning logs | 30 days | Auto-deleted |
| User account data | Until account deleted | Account deletion request |
| Stripe billing data | 7 years | Regulatory requirement — cannot be deleted early |

## Data Deletion

### Delete a project

Deleting a project permanently removes:
- All code chunk embeddings for that project
- All analyses, agent runs, outputs, and citations
- GitHub token for that project

```
DELETE /api/v1/projects/:id
```

Deletion is cascaded via `ON DELETE CASCADE` in PostgreSQL — all associated data is removed atomically.

### Delete your account

```
DELETE /api/v1/users/me
```

This deletes:
- Your user record
- All projects and their data
- All analyses and their outputs
- Your Clerk identity

What remains:
- Stripe billing records (retained for regulatory compliance, cannot be deleted)
- Anonymised usage analytics (no PII, cannot be linked to you)

### GDPR Right to Erasure

Submit a deletion request via:
- Account settings → Delete Account, or
- Email: privacy@archon.ai

We process requests within 30 days and send confirmation when complete.

### GDPR Data Portability

Export all your data at any time:
- Account settings → Export Data
- Or via API: `POST /api/v1/users/me/export`

Includes: all projects, analyses, outputs, and citations in JSON format.

## Security of Data at Rest

- PostgreSQL database: encrypted at rest by Railway (AES-256)
- GitHub tokens: encrypted with application-level Fernet encryption before DB storage
- Redis: in-memory only, no persistence to disk (data is ephemeral job state)
- All Railway volumes: encrypted at rest

## Security of Data in Transit

- All external communication: TLS 1.3 minimum
- Internal service communication: TLS within Railway private network
- Database connections: SSL required (`sslmode=require`)
